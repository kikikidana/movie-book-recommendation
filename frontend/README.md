# Frontend - Movie & Book Recommendation Service

## 📋 개요

React 기반의 사용자 인터페이스입니다.

## 🚀 시작하기

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm start
```

브라우저가 자동으로 [http://localhost:3000](http://localhost:3000)에서 열립니다.

## 📁 폴더 구조

```
src/
├── components/     # 재사용 가능한 UI 컴포넌트
├── pages/          # 페이지 컴포넌트
├── services/       # API 호출 함수
├── App.js          # 메인 App 컴포넌트
├── index.js        # 진입점
└── index.css       # 전역 스타일
```

## 🎨 주요 페이지

- **Home** - 서비스 소개
- **Recommend** - 추천받기 (취향 입력)
- **Login** - 로그인
- **Signup** - 회원가입
- **MyPage** - 추천 기록, 즐겨찾기

## 📦 사용 패키지

- **react** - UI 라이브러리
- **react-dom** - React DOM 렌더링
- **react-router-dom** - 라우팅
- **axios** - HTTP 클라이언트

## 🏗️ 빌드

```bash
npm run build
```
